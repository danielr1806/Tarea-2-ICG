Para esta tarea se usaron las bibliotecas de Freeglut, GLEW y AntTweakbar.
Se puede usar la interfaz tanto via AntTweakBar como por teclado.
Explicación de la interfaz:
- Menú:
 - Dibujar curva: Se dibuja la curva clickeando la pantalla. Para indicar el fin de una curva, se puede presionar la tecla "ENTER" o se puede cambiar de opción en Menu.
 - Seleccionar: Muestra el BoundingBox de la curva seleccionada. Se puede transladar la curva clickeando y arrastrando el mouse a su nueva posición.
 - Puntos de Control: Muestra los puntos de control de la curva. Esta opción debe estar seleccionada para hacer operaciones sobre los puntos, como Editar, Eliminar o Elevar Grado.
 - Cambiar Color: Cambia el color de una curva seleccionada.
 - Subdividir: Se hace clic en cualquier curva previamente seleccionada y esta se divide en 2 curvas nuevas.
- Puntos de Control:
 - Editar: Cambia la posición de un punto de control arrastrándolo con el mouse a su nueva posición. La opción "Puntos de Control" debe estar seleccionada en el Menú.
 - Eliminar: Se hace clic en el punto de control deseado y se elimina. La opción "Puntos de Control" debe estar seleccionada en el Menú.
- Color Curva: La opción "Cambiar Color" debe estar seleccionada en el Menú.
- Ángulo de Rotación: Se indican el ángulo con el que se desea rotar.
- Rotar: La opción "Seleccionar" debe estar seleccionada en el Menú. También se puede usar la tecla "ENTER".
- Elevar Grado: La opción "Puntos de Control" debe estar seleccionada en el Menú. También se puede usar la tecla "+".
- Aumentar Tamaño: La opción "Seleccionar" debe estar seleccionada en el Menú. También se puede usar la tecla "+".
- Reducir Tamaño: La opción "Seleccionar" debe estar seleccionada en el Menú. También se puede usar la tecla "-".
